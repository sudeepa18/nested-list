Understanding Nested Lists

A nested list is a list inside another list.

Unordered Nested List

A nested unordered list is a list where each item has sub-items , all marked with bullet points

  • Fruits
    • Apple
    • Banana
    • Orange
  • Vegetables
    • Carrot
    • Broccoli
    • Spinach

Ordered Nested List

A nested ordered listed is a list where items are numbered, and sub-itms also follow a numbering system.

  1. Frontend Technologies
    1. HTML
    2. CSS
    3. JavaScript
  2. Backend Technologies
    1. PHP
    2. Python
    3. Java
